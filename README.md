## 1. 概述
这是C++的入门教程，是我在学习C++的过程中，将所用的英文版教程翻译的中文版，一来希望提高自己的水平，二来可以供所需之人观看。本文全是出于兴趣而翻译，不敢保证内容的绝对精确性，仅仅是尽力而为，若大家发现错误，欢迎指出。
